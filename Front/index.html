Formação Acadêmica

Formado em Engenharia de Computação pela Universidade Federal de Itajubá (Unifei), Campus Itabira, em 04 de agosto de 2017. Tem interesse e participa de atividades relacionadas à tecnologia, software e hardware.

Iniciação científica

Desenvolveu um sistema de aquisição de sinais eletromiográficos via Bluetooth, para exibir o sinal processado em tempo real em uma tela de computador. Foram utilizados os microcontroladores PSoC 5LP, para conversão analógico-digital de sinais e filtragem de ruídos, e PSoC 042 BLE, para transmitir o sinal via Bluetooth.

Equipe de robótica

Foi programador na equipe de robótica Drumonters da Unifei, Campus Itabira. Colaborou para a programação de MindStorm e desenvolveu software de processamentos de imagens para o futebol de robôs utilizando a linguagem de programação C/C++ com auxílio da biblioteca OpenCV.

Coliseu

Criou uma inteligência artificial híbrida chamada Coliseu, utilizada para resolver problemas de otimização. A ideia inicial do Coliseu consiste em utilizar pelo menos dois algoritmos evolucionários, como por exemplo PSO e GA, para competirem entre si.
