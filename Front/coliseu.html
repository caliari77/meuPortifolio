Coliseu - O que é?

É um algoritmo heurístico, que teve seu nome inspirado nas lutas entres gladiadores do Coliseu, que aconteciam na Roma antiga. Recebeu esse nome, pois utiliza da competição entre pelo menos dois algoritmos heurísticos, chamados de gladiadores, para tentar otimizar um problema real.

Como funciona?

Cada gladiador funciona separadamente, apresentando uma solução para o problema. Os resultados são comparados e o algoritmo com melhor solução é o gladiador vitorioso no turno. Depois, é dado início a um novo turno, porém com a solução anterior como sugestão para o turno atual. O procedimento é repetido por quantos turnos o programador achar necessário.

Qual a sua origem?

Sob a orientação do professor Sandro Carvalho Izidoro, implementei o Coliseu como Trabalho de Conclusão de Curso. Ele foi utilizado para resolver o problema de estabilização de drones com quatro motores. Foram realizados diversas simulações computacionais para otimizar os ganhos de um conjunto de controladores PID. No trabalho foram testados os algoritmos já amplamente conhecidos PSO e GA. A partir desses algoritmos evolucionários, bastante difundidos dentro de computação natural, crio-se o Coliseu.
